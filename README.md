# Sniffies Hax4Life

A modern React-based mobile application built with Capacitor for cross-platform deployment.

## 📱 About

Sniffies Hax4Life is a feature-rich mobile application that combines modern web technologies with native mobile capabilities.

## ✨ Features

- 🎨 Beautiful UI with Radix UI components
- 📊 Data visualization with Recharts
- 🎯 Form handling with React Hook Form
- 🌙 Dark/Light theme support with next-themes
- 📱 Cross-platform mobile support via Capacitor
- 🔄 Real-time data fetching with TanStack Query
- 🎭 Accessible components with proper ARIA attributes
- 📦 TypeScript for type safety

## 🚀 Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn
- JDK 17+ (for Android builds)
- Android SDK (for APK compilation)

### Installation

```bash
# Clone the repository
git clone https://github.com/ThunderConstellations/sniffies_hax4life.git
cd sniffies_hax4life

# Install dependencies
npm install
